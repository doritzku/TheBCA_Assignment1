# Yahoo Finance API Data Retrieval

This Python script fetches stock market data from the Yahoo Finance API using RapidAPI. It retrieves data for a specified symbol, interval, and range, and optionally compares it with other market indices.

## Requirements

- Python 3.x
- `requests` library (to make HTTP requests)
- `colorama` library (for colored terminal output)

You can install the required libraries using pip:

```bash
pip install requests colorama
